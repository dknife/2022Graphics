# 2022Graphics

# 3D 컴퓨터 그래픽스
#### 이 저장소는 2022년 동명대학교 게임공학과 그래픽스 프로그래밍 수업을 위한 저장소입니다.
#### 이 저장소의 코드는 Visual Studio Code + Python 3 + PyOpenGL + PyQt5 로 테스트되었습니다.
